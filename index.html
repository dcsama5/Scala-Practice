Built in Control Structures

Breaks and continues

There are no breaks and continues in scala. Use boolean variables to automatically cut from a loop.

{% highlight scala %} import scala.util.control.Breaks._; import java.io._; val in = new BufferedReader(new InputStreamReader(System.in)) breakable { while(true) { println("?") if(in.readLine() =="") break } } {% endhighlight %}
